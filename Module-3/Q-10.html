How can we club two or more rows or columns into a single row or column in an HTML table? With Example. In HTML, it is not possible to merge multiple cells into one, but you can give the impression that you can by utilizing the colspan and rowspan attributes. colspan: This property indicates how many columns a given cell should span. It is positioned inside the cell you wish to enlarge's opening tag. rowspan: This property indicates how many rows a given cell should span. It is positioned inside the opening tag, just like colspan. Example:
