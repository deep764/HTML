How can we club two or more rows or columns into a single row or column in an HTML table? With Example. You can't actually combine cells into a single cell in HTML, but you can create the illusion of merging cells using the colspan and rowspan attributes. colspan: This attribute specifies the number of columns a cell should span. It's placed within the opening tag for the cell you want to expand. rowspan: This attribute specifies the number of rows a cell should span. Similar to colspan, it's placed within the opening tag.
