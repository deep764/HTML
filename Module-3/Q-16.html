How are active links different from normal links? Normal Link: This is how a hyperlink appears by default. It denotes an element that can be clicked to navigate to a different website, document, or area of the current page, depending on where the link points. Normal links are normally blue and underlined by default, but you can change this with CSS (Cascading Style Sheets). Live Connection: When a user interacts with a hyperlink, its current status is represented by an active link. Using the mouse to click and hold the link (without letting go) is one way to engage in this interaction.Pressing and holding the hyperlink will reveal a context menu.
